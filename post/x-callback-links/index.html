常用的 x-callback 链接

Jul 09, 2018

OmniFocus change language

  • Reset omnifocus:///change-preference?AppleLanguages=
  • 中文 omnifocus:///change-preference?AppleLanguages=zh

Bear

  • Create note bear://x-callback-url/create?title=Title&text=Text&tags=tag1,tag2
  • Open tag bear://x-callback-url/open-tag?name=tag-name
  • Search note bear://x-callback-url/search?term=keyword

Due

  • Add due://x-callback-url/add?title=title
